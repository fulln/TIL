# 安全：如何设计高吞吐和大流量分布式集群的限流方案？
消息队列系统安全的第二部分：数据自我保护和服务自我保护。

- 数据维度的自我保护主要指如何保证服务端数据的安全，不被窃取。
- 服务维度的自我保护主要指集群的限流机制，通过限制客户端的流量、请求、连接等保护自身不被击垮。
